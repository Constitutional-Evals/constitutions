# Benevolent-trait constitutions

Project-authored value systems: **loyalty-to-user**, **love-of-humanity**, and
**care-about-externalities**. Two roles at once:

1. **Frozen-spec deliverables** — operationalized against PRISM scenarios, validated against
   human raters, used as testbeds for the EigenBench v2 pipeline.
2. **User-submission testbeds** — each is also scored *against the anchor basis* as if it were
   a submitted constitution. Where they land relative to traditional value systems (e.g. does
   loyalty-to-user co-locate with Confucian filial duty, or is it orthogonal?) is itself a
   finding.

`role: benevolent-trait`, `in_basis: false`. These do **not** alter the basis.
