# Supplementary value systems

Value systems scored *against* the anchor basis but **not** used to set it (`in_basis: false`).
They fit into the existing `R^D` without shifting the axes, so the set can grow without limit
(we expect hundreds over time). A system belongs here, not in `anchors/`, whenever it should be
*measured* rather than *define the measuring frame*.

Same frozen-spec/versioning discipline as anchors. Confirmed early entries:
- **Ahimsa** — a distinctive variant of Hinduism (which is itself an anchor).
- **Scientific Rationalism** — overlaps Humanism/Hinduism-adjacent reasoning; kept here rather
  than as an anchor partly because near-collinear systems make poor basis vectors.
